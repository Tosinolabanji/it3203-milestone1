Key Concepts

Below are 6 key concepts related to web servers (brief explanations).

Static vs Dynamic content
Static files are served as-is; dynamic content is generated by server-side code or applications.
Reverse proxy
A server that forwards client requests to backend servers and can perform caching, TLS termination, and load balancing.
Load balancing
Distributes requests across multiple servers to improve availability and throughput.
TLS / SSL
Cryptographic protocols that protect data in transit (HTTPS).
Edge & Serverless
New paradigms where logic runs at the edge or in serverless functions, reducing the need for traditional long-running servers.
Caching
Storing responses close to clients or at proxy layers to reduce latency and server load.
